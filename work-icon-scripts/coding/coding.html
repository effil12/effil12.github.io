The Eggcelent Warrior

The Eggcelent Warrior is a 2D game that I and 4 other people made. It was created by using Unity engine. My responsibility was the Unity development which mean the layout designing of the scenes, menus and programming in C#.

The game is about a chicken who has lost it's rooster and has to fight through 5 dimensions to save him. With every dimension being harder than the previous.
